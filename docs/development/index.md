## Development Standards

- **Code Quality**: 100% test coverage, zero `any` types
- **Architecture**: SOLID principles, clean separation of concerns
- **Documentation**: JSDoc for all public APIs, comprehensive guides
- **Testing**: Unit + Integration + Mutation testing
- **Git**: Conventional commits, feature branches
